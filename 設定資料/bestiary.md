# 生物・マナ生命体 図鑑（Bestiary）

> 本編設定資料（game_design_bible.md）の付属資料。§11「マナ生命体」の詳細版。

## 凡例
- **属性**：火／水／風／土／無。**無属性は特定の弱点を持たない**（相性ダメージが乗らない）＝バースト討伐しにくい。ボス・レア・特殊個体に多い。
- **階級**：低級（知能・能力ともに低い）＜ 中級 ＜ 上級 ＜ ボス級。マナを取り込むほど強く・賢くなるため、同じ系統で進化・成長ラインを成すことが多い。
- **(レア)**：出現率が低く、ジェム・素材を多く落とす。

## C. ステータス／レベル／報酬の設計（共通モデル）

> 数値はすべて**暫定（バランス調整フェーズで確定）**。モンスターのLvは出現域の難度目安も兼ねる。

### C-1. プレイヤーのレベル
- レベル帯：**Lv1〜99**（提案）。レベルアップで基本6ステータス（§4）が伸び、より重い魔法の発動が現実的になる（§6：全魔法は入力可能だがLv・ステ不足だと発動率が下がる）。
- 必要経験値（暫定式）：`次レベルまで = round(15 × Lv^1.8)`
  - 目安：Lv1→2 ≈ 15／Lv5→6 ≈ 270／Lv10→11 ≈ 950／Lv20→21 ≈ 3,300／Lv50→51 ≈ 17,900／Lv99 ≈ 63,000

### C-2. モンスターのステータス項目
| 項目 | 意味 |
| --- | --- |
| Lv | レベル（出現域の難度目安） |
| 属性 | 弱点判定（無は弱点なし） |
| HP | 体力 |
| 基本ステータス | DEX/AGI/INT/PIE/VIT/LUK。敵の攻撃性能・耐久・移動・詠唱・報酬計算の基礎 |
| 経験値(EXP) | 撃破時に得る経験値 |
| ジェム(G) | 撃破時に分解されて得る通貨（1G≒1円／§14.5） |
| ドロップ素材 | 撃破時に確率取得（%） |
| 特性 | 行動・逃走・召喚など |

### C-3. ドロップ率の目安
- コモン素材 30〜60%／アンコモン 10〜25%／レア 3〜8%／超レア（ユニーク素材）0.5〜2%
- レアモンスター自体の出現率：0.5〜3%
- 現行仕様：全モンスターに低確率で**レアカラー個体**（金／虹）が出現する。EXP・ジェム・ドロップ率が上昇し、ミニボーナス敵として扱う。

## F. 設計メモ
- **無属性の役割**：弱点を突けない（相性ダメージが乗らない）ため、バフ→大魔法のバースト討伐がしにくい。ボス・レア・特殊個体に集中させることで、終盤やレア狩りの歯ごたえを担保する。
- **成長ライン＝マナ捕食**：「マナを取り込むほど強く・賢くなる」設定により、低級→上級→ボス級が同系統の進化・成長として自然に並ぶ（例：マナスライム→エリクサースライム、ゴブリン→ホブゴブリン→ゴブリンロード）。
- **魔人寄りの個体**（ハーピー、セイレーン、アルラウネ等）：魔物と魔人の境界はグラデーション。知能の高い個体は §11「種族・魔人」に接続し、一部は敵対せず共存する。
- **未割り当て**：各種のドロップ素材、HP/攻撃などの具体数値、出現領（§15）との対応は実装フェーズで設定。
