# タイピング魔法アクションRPG ― 設定資料 (Game Design Bible)

> **作品タイトル：AMC（A Mana Circulation）**
> ※「AMC」は二重の意味を持つ。**プレイヤー向けのタイトル＝「A Mana Circulation」**（循環というテーマ）。一方、**本編の用語としての「AMC」＝人工マナ生命（Artificial Mana Creature）**を一貫して指す。
> ※演出：賢者との邂逅で古代史を理解した後、タイトル画面が **「Another Mana Circulation」** へと変わる（循環の暗示）。
> ジャンル：タイピング × 魔法アクションRPG／2Dトップダウン・ピクセルアート
> 最終更新：設定固め完了時点

## 0. ひとことコンセプト

**「呪文を"打つ"ことで魔法を発動する、王道ファンタジーのアクションRPG」**

プレイヤーはローマ字で呪文を入力して魔法を放つ。小さな魔法でバフを積み上げ、大魔法で爆発させ、撃つとバフが全解除されてまた積み直す――この**「積み上げては崩れて元に戻る」戦闘ループ**が、そのまま世界の歴史テーマ（マナ文明は発展すると必ず崩壊し、同じ地点へ戻る循環構造）と重なっている。

タイピング速度・精度・立ち回りが、すべて火力に直結する。

## 1. コアゲームプレイ

### 1.1 操作（通常モード）

| 入力 | 動作 |
| --- | --- |
| **WASD** | 移動。敵の攻撃を避ける |
| **Shift** | マナバリア（ガード）。押している間、時間経過で少しずつマナを消費。被弾時はマナを**10%消費**して被ダメージを**75%カット**。敵の攻撃にタイミングを合わせると**パリィ**成立 → 自分の周囲の敵を**5秒間スタン**し、さらに**マナを25%回復**する |
| **Ctrl** | 短距離前方テレポート（画面半分程度）。回避用。**短いクールタイムあり／マナ消費なし**（マナ0でも使える最低限の回避手段） |
| **Space** | 詠唱準備モード ⇄ 通常モードの切り替え |

### 1.2 操作（詠唱準備モード）

- プレイヤーは**移動できず**、ローマ字入力のみ可能（＝完全に無防備）。
- 入力 → **Enter** で確定 → MP消費 → 詠唱開始 → 詠唱時間経過 → 発動。

**入力例（ファイア）**
`Space`（詠唱準備へ）→ `faia` と入力 → `Enter` → MP消費 → 詠唱 → 発動

### 1.3 タイピング判定

- **Backspaceで修正可能。**
- **ミスタイプしたままEnterすると、何も起こらない**（MPも消費しない）。
- ローマ字の表記ゆれは許容する。例：`tya` / `cha` / `tixya` はどれも「ちゃ」。現実のローマ字入力と同じ挙動。

### 1.4 詠唱中断ルール

詠唱中に、**累計で最大HPの25%以上の被ダメージ**を受けた時点で、強制的に詠唱準備状態が解除される。

> 含意：詠唱中は「25%のダメージ枠」を盾に時間を稼いでいる状態。小ダメージなら耐えて押し通せるが、大技や複数ヒットで一気に25%を超えると、MPも詠唱時間も無駄になる。後述の「被弾時マナ消費で被ダメ軽減」が、この枠を長持ちさせる役割を持つ。

### 1.5 リソース

#### HP（生命力）
- 0になると**ゲームオーバー＝強制帰還**。
- 自動回復量は以下で決まる：
  - **現在のMPの割合**（MP満タンで毎秒5%、MP0で回復なし）
  - ＋装備による追加
  - ＋スペルのバフによる追加

#### MP（マナポイント＝マナ。攻防で共有する単一リソース）
- スペルの詠唱、ガード（Shift：押している間に微減＋被弾時に10%消費）などで消費。**パリィ成功時はマナを25%回復**（§1.1参照）。
- 自動回復量は以下で決まる：
  - レベルに応じた基礎回復量
  - ＋PIEによる追加
  - ＋装備による追加
  - ＋スペルのバフによる追加

#### 拡張MP（マナ親和：撃つほど器が広がる）
- 小スペルを詠唱するたび、その**消費MP分だけ最大MPが一時的に増える**（例：ファイア消費10 → 最大MP+10）。**バフと同じ継続時間**で、増えたぶんは時間経過で消える。
- **拡張の上限は「初期最大MPと同量（+100%）」＝合計で初期最大MPの2倍が天井。** 天井に達すると、それ以上小スペルを撃っても拡張は増えない（バフは引き続き乗る）。→ これにより**初期最大MPを上げる育成（INT・装備）が天井の底上げとして価値を持つ**。
- **増えるのは最大値だけ。現在MPは増えない**——自動回復などで満たして初めて使える（実質的なマナ回復ではない）。最大MPが縮むと、超過した現在MPは切り捨てられる。
- 役割：積み上げ中に**マナの器を広げ、高コストの大魔法を撃てるようにする**（火力には直結させない＝暴走防止）。
- **大魔法を撃つと、全バフと同時に拡張MPも解除**され、最大MPは初期値に戻る。大魔法自体への威力ボーナスは無し。
- 世界観：スペルを撃つ＝マナとの親和性を高め、扱えるマナの器が一時的に広がる（§13）。大魔法で一気に放出して親和をリセット。
- UI：拡張ぶんはMPバー右端に**虹色（高濃度マナ）**で描画（§9.2）。

> **設計の核：攻めるほど脆く、守るほど固い。**
> HP回復が「現在MP割合」に依存するため、マナを使い切って大魔法を連発すると火力は出るがHP回復が止まって脆くなる。マナを残して立ち回ると火力は控えめだがHPがじわじわ回復して粘れる。MPは攻撃と防御で取り合う単一リソースなので、強い魔法（高コスト）を撃った直後が最も無防備になる。

### 1.6 戦闘計算（ダメージ・相性・クリティカル・状態異常）

> 数値・係数は**暫定**（実機で動かして調整する方針）。

**ダメージ計算式（暫定）**
```
最終ダメージ = max(1, 魔法攻撃力 × 威力係数 × バフ倍率 × クリティカル × 相性 × 敵耐久補正)
```
- **魔法攻撃力** = INT + PIE×0.8 ＋ 装備・バフ補正
- **威力係数**（スペルの威力ランク）：小0.8／中1.2／大2.0／特大3.5
- **バフ倍率** = 威力アップ系バフの合算（例 1 + Σ威力+%）
- 敵耐久補正はVIT等の基本ステータスから算出する。独立したATK/DEF値は持たない。

**相性倍率**（ダメージ表示色と連動）
- 弱点を突く＝**×1.5**（赤）／通常＝**×1.0**（白）／相性で軽減＝**×0.7**（灰）
- 4元素の循環：火→風→土→水→火。**敵の弱点属性で攻撃すると×1.5**、敵が得意な属性（敵に有利な側）で攻撃すると×0.7、それ以外（同属性含む）は×1.0。
- **無属性の敵は弱点を持たない**ため、常に×1.0。

**クリティカル（暫定）**
- 基準クリティカル率 = 5% ＋ LUK寄与 ＋ 装備。クリティカル時ダメージ = **×1.5** ＋ DEX寄与（クリダメ）＋ 装備。式の `クリティカル` 項に乗算で反映。

**状態異常（暫定）**
- 各属性の攻撃が、対応する状態異常の**蓄積値**を敵に与える（基礎 約10〜30／ヒット。ルーン「アフリクト」や状態異常蓄積バフで増加）。
- 蓄積が**閾値（例100）**に達すると発動。発動後は一定時間効果、時間経過で減衰。
  - **火傷（火）**：発動中、敵の**被ダメージ+25%**＋毎秒小ダメージ（約10秒）。
  - **出血（風）**：敵が**行動（移動・攻撃）するたびに固定ダメージ**（約10秒）。
  - **鈍化（水）**：敵の**移動・攻撃速度-40%**（約8秒）。
  - **気絶（土）**：蓄積が満ちると一定確率で**行動不能**（約2秒）。
- ※プレイヤー側はこれらの状態異常を受けない（敵専用／§14.7・消費アイテム方針と整合）。

## 2. 魔法システム

魔法は **スペル** と **ルーン** という、生まれた背景の異なる2つの概念から構成される。

### 2.1 スペル（Spell）

- 魔法の**現象そのもの**を指定する。例：ファイア＝火球を射出。
- **古代文明（厳密には古代のAMC）が作った技術**であり、今は失われている。存在するものは使えるが、現代人には内容が理解できず、新しく作ることもできない。
- **1詠唱につき1個まで。**
- 収集対象（有限の宝物）として機能する。

### 2.2 ルーン（Rune）

- スペルの現象に**変化を加える**ための、**現代人が後から生み出した制御技術**。
- 例：
  - **ダブル（double）**：消費MPが2倍になる代わりに、一定時間後に修飾したスペルをもう一度発動する。
  - **レンジド（ranged）**：飛距離を伸ばす。
- 1詠唱にいくつでも詰め込めるが、詠唱難度が上がり、消費マナが増える。

**比喩**：スペルが銃そのもの。ルーンが銃身を削る／長くする／連射する／弾を変える、といった加工。

### 2.3 詠唱の構文

```
[ルーン … 複数・順不同] + [スペル … 最後に1個]
```

例：`double ranged faia` ＝「2回発動する・長射程の火球」
（`ranged double faia` でも結果は同じ。ルーンの順番は自由）

### 2.4 コスト（乗算）

ルーンは消費マナを**乗算**で増やす。

例：威力UP（×2）＋ 範囲UP（×1.5）＋ 継続時間UP（×2）をファイアに乗せると
**基礎MP × 2 × 1.5 × 2 ＝ 6倍**

- 乗算なので盛るほど**指数的**に重くなり、自然に上限がかかる（際限なく詰めると詠唱が成立しない）。
- 加えて**詠唱文字列が長くなる**ため、無防備時間が伸び、ミスタイプ率も上がる。マナ的制約とタイピング的制約の**二重の罰**がかかる。
- **スペルのバフ効果は、ルーンと違ってMP消費を増やさない**（バフ＝発動後の状態、ルーンの乗算＝詠唱時のコスト、と層が別）。
- **同種ルーンの重複は不可**（1詠唱につき同じルーンは1枚まで）。同じルーンを重ねて入力しても効果は重複しない。

### 2.5 バフ・積み上げシステム（戦闘の背骨）

スペルには**メイン効果**（その場の攻撃・回復など）と**バフ効果**の2種類がある。

- 例：
  - **ファイア**（弱い基礎スペル）→ 威力アップ・**小さいが長い**バフ（土台になる）。
  - **エクスプロード**（強いスペル）→ 威力アップ・**大きいが短い**バフ。
  - **メテオ**（大魔法）→ 威力は高いが、撃つと**すべてのバフを解除する**。
- **効果量と継続時間は反比例**：弱い魔法ほど効果量は小さいが継続時間が長く、強い魔法ほど効果量は大きいが継続時間が短い。→ 弱→強へ自然にエスカレートし、短命な強バフが切れる前に大魔法でフィニッシュする流れが生まれる（具体値は spells_and_runes.md）。
- **同じスペルのバフは重ねがけされない。効果時間をリフレッシュするだけ。**
  - → 多くのバフを同時に維持するには、種類の違うスペルを次々に打つ必要がある。
- **異なるスペルのバフの同時維持数に上限はない。** ただし、すべてのバフに継続時間があり、各スペルの発動には詠唱準備＋詠唱時間がかかるため、入力を限界まで速くしても**無限に積むことはできない**（時間という自然な上限がかかる）。
- 積み上げと並行して、小スペルは**最大MP（拡張MP）**も同じリズムで一時的に押し上げる（§1.5）。**バフ＝火力の土台／拡張MP＝マナの器**の二本柱が同時に育ち、どちらも継続時間で消え、大魔法で一括解除される。

**大魔法（だいまほう）**
- 一部のスペルが持つ性質。様々な種類がある。
- 「大魔法」という名前だが、**そのまま発動しても威力は低く、詠唱は長く、使いにくい**。バフを積み上げた状態で撃って初めて真価を発揮する。
- **大魔法に該当するスペル**：メテオ（火）／ストーム・ライトニング（風）／タイダルウェーブ・ダイアモンドダスト（水）／ガイア・ジャベリン（土）。
- **大魔法自身はバフ効果を持たない。撃つとすべてのバフ＋拡張MP（§1.5）を解除する（フィニッシャー専用）。**

**戦闘ループ**
```
小スペルでバフを積む（威力UP・詠唱短縮など）
   ↓ 重ねるほど次が速く・強くなる（ウィンドの詠唱短縮で加速する正のループ）
大魔法で全バフを乗せて一撃 → バフ全解除でリセット
   ↓
また小スペルから積み直し
```

> **タイピングが速いほど、バフ解除でリセットされる前に多くのスペルを重ねられる＝より強力な大魔法を放てる。**
> タイピング上達がそのまま火力に直結する。これがタイピングゲームである必然性の核。

### 2.6 ダメージ構造

- 小魔法は1発の火力は低いが、積み上げフェーズで多数撃つため、**当て続けると総ダメージに効く**。外すとバフは付いてもダメージを捨てることになる。
- 大魔法が**メインのダメージソース**。
- 総ダメージ ＝ **小魔法の積み重ね（手数 × 命中）＋ 大魔法の一撃（バフ全乗せ）**
- → 小魔法は「バフを積む」と「地味に削る」の二役。**速く・正確に・当てながら**積むのが理想で、タイピング速度・精度・立ち回りのすべてが報われる。

### 2.7 属性（リゾーマタ四大元素ベース）

各属性に大まかな方向性がある。

| 属性 | 方向性 | 状態異常（敵に付与） |
| --- | --- | --- |
| **風** | 詠唱速度 | **出血**：くらうほど、移動・攻撃などの行動ごとにダメージを受ける |
| **火** | 火力 | **火傷**：くらうほど被ダメージが増える |
| **水** | 再生（回復・リソース維持） | **鈍化**：くらうほど攻撃・移動などが全体的に遅くなる |
| **土** | 防御 | くらうほど**気絶**する確率が高まる |

> 設計メモ：水は方向性が「再生・支援」だが状態異常は「鈍化（妨害）」で攻守の顔が異なる。**全体としては「支援寄り」に振る**（回復・MP維持・継続戦闘を主軸とし、鈍化はあくまで補助的な妨害）。状態異常はすべて敵に与えるもの。

### 2.8 属性相性

```
火 → 風 → 土 → 水 → 火 …（各属性が、次の属性に強い）
```

- 火には水が効く（水 > 火）
- 水には土が効く（土 > 水）
- 土には風が効く（風 > 土）
- 風には火が効く（火 > 風）

**運用上の意味**
- **小魔法（積み上げフェーズ）**：火力に直結しないため、属性相性を気にせず**何でも自由に**積む。速く正確に多くの種類を打つことが正義。
- **大魔法（フィニッシュ）**：敵の弱点属性を突く必要がある。→ 「同じ属性しか使わない」状況を防ぎ、複数属性の大魔法を揃える動機になる。

> プレイヤーの思考は二段階に分かれる。「今は何でもいいから速く積む」フェーズ（タイピング地力）と、「敵の属性を見て対応する大魔法を選んで撃つ」フェーズ（属性ビルドの幅と判断）。手持ちの大魔法を4属性ぶん揃えておかないと弱点を突けず詰むため、大魔法こそ収集・育成の的になる。

## 4. ステータス

### 4.1 基本6ステータス

| ステータス | 影響するサブステータス |
| --- | --- |
| **DEX** | 移動速度／攻撃範囲／クリティカルダメージ |
| **AGI** | 詠唱速度／移動速度／回避率 |
| **INT** | MP量／魔法の攻撃力／経験値獲得量 |
| **PIE**（piety） | 魔法攻撃力／MPの自動回復量／HPの自動回復量 |
| **VIT** | HP量／防御力／アイテム所持重量 |
| **LUK** | クリティカル率／回避率／アイテムドロップ率 |

> ステータスの役割重複は**意図的**。移動速度＝DEX＋AGI、回避率＝AGI＋LUK、魔法攻撃力＝INT＋PIE のように複数ステに分散しており、どのビルドでも複数ステを伸ばす意味がある（一点特化を防ぎ、装備厳選を促す）。
> 特に **AGIの「詠唱速度」** は、ゲームの核（速く積む→大魔法強化）に直結する実質的な花形ステ。

### 4.2 ビルドの方向性（目安）
- DEX：機動・範囲・クリダメ（立ち回り型）
- AGI：詠唱速度・回避（積み上げ速度＝火力に直結）
- INT：MP量・魔法火力・経験値（純魔法アタッカー）
- PIE：魔法火力・MP/HP回復（持続戦闘型）
- VIT：HP・防御・所持重量（耐久）
- LUK：クリ率・回避・ドロップ率（運用＆収集効率）

### 4.3 ステータス振り分け・振り直し
- レベルアップで**ステータスポイント**を獲得し、プレイヤーが6ステータス（DEX/AGI/INT/PIE/VIT/LUK）へ**自由に振り分ける**（ビルドの自由度の核）。
- **振り直し**：ギルドで**ジェムを支払う**ことでリセット可能（費用はレベル・回数に応じて増加）。

## 5. 装備

### 5.1 装備枠（計6枠）
- 武器 ×1
- 服 ×1
- 帽子 ×1
- 指輪 ×2
- ネックレス ×1

### 5.2 ステータス構成
各アイテムは以下を持つ：
- **固有の基礎ステータス 2個**
- **レア度に応じたランダムステータス 1〜3個**

### 5.3 ランダムステータスの種類
- 基本6ステータスの**実数値**を増やす
- 基本6ステータスを**%**で増やす
- 6ステータスによって増える**サブステータス**を直接増やす
- **特定のスキルのダメージ**を増やす

> 「実数で底上げ」「%で乗算」「サブ直撃」「特定スペル特化」という複数の厳選軸があり、ハクスラとして十分な深みを持つ。`特定スキルのダメージ+` により、お気に入りの大魔法を中心に据えたビルドが組める。

## 6. 死亡・ペナルティ・周回構造

- HP0 → 魔道具により**即座にギルドへ強制帰還**。即死の場合は蘇生できない。
- **死亡時は治療費としてお金（ジェム）が減る。**
- **集めかけのジェム・素材は持って帰れる。**
- **ダンジョンはリセット**（毎回入るたびに中身が変わる。後述「別世界線」設定による）。
- **全魔法は最初から入力可能**。ただし**MPが足りなければ撃てず、詠唱時間も長い**ため、序盤は高コストの大魔法を実用しづらい（自然な制限）。
- **発動の失敗（不発）はしない**——練度による失敗ストレスを避ける。代わりに、**威力が高いほど魔法のエフェクトが派手になる**（同じ魔法でも、ステータス・バフ・練度が上がるほど見た目が豪華になり、上達が視覚で報われる）。

## 8. ダンジョンと現世（オーバーワールド）

### 8.1 現世（固定）
- マップ・敵ともに**固定**。ここでファーム（周回）することもできる。
- **スペルは現世に存在する**（AMCが現世に作った石碑から習得する）。ダンジョン内にはない。

### 8.2 ダンジョン（手続き的生成）
- **AMCの作った遺構**。入るたびに中身が変わる。
- 設定上の理由：AMCの超技術により、**入るたびに別の世界線に移動している**。
- 起源：初期のAMCたちの資源回収手段。有限だったマナを無制限に得るため、**別の世界線からマナを取ってくる技術**を編み出した。後に別惑星からより大量のマナを集められるようになり、あまり利用されなくなった。
- 目的：ジェム・素材稼ぎ、依頼。
- **AMC遺構（ダンジョン）は各領に点在する**（規模は小〜大さまざま／§15参照）。最奥・最大規模の **忘却の遺跡群（小陸・アウターエッジ領）はエンドコンテンツ**。
- **難易度選択**：各ダンジョンは挑戦時に難易度を選べる（敵Lv・報酬・レアドロップ率が変動）。

## 10. マナ

### 10.1 概念
- 現代における**水のように、どこにでも普遍的に存在する**。燃料であり、資源であり、人類及び生物の根幹。
- マナによって生命維持を行う**マナ生命**や、マナを他のエネルギーに変換する**スペル**がある。
- 「マナとは何か」＝**マナはマナ**。この世界では水が生命の源として普遍的に在るのと同様、当たり前でありふれた物質であり概念。説明はナンセンスになる。
- **超高密度の液体状マナ＝マナエリクサー**（かつては伝説上の存在とされていた）。
- **固体化した物質＝マナジェム**（生活に利用される）。
- 通貨としても使用。単位は **G（mana Gem、ジェム）**。

### 10.2 物理法則（重要・ギミックに直結）
- マナは**負の質量（負の重力）**を持つ。
- ただし物質にマナが宿っている場合、マナと物質の重さで相殺され、軽い物質にはなるが**マイナスにはならない**。
- 濃度が高くなるにつれ、重さが反転して**浮遊する**。
  - → 飛空艇の浮力原理。エリクサー（高密度＝強い負の重力）は運搬・動力にも関わる。
  - → ラスボス「賢者の石」の反重力・時間加速ギミックの根拠。

### 10.3 色（アート・UIに直結）
- マナは本来すべての属性を含有しているため、**白色・虹色・プリズム**のイメージ。
- 低濃度では色が宿っている物質に依存するため、**MPバーは体内に流れるマナから青色**で表現。
- **マナジェム（G）**＝結晶化したもの。**虹色に輝くムーンストーン**のような見た目。
- **エリクサー**＝白色で、角度によって様々な色に見える**虹色の遊色**。
- **魔法エフェクト**＝属性色（火＝赤、風＝緑、水＝青、土＝茶）。**高位魔法ほど白・虹みが混じる**（高濃度＝全属性含有＝白虹のロジック）。
- **賢者の石**＝白虹のパーティクルをまとう。ラスボスだけ色の格が違う演出。

## 11. マナ生命体（通称：魔物）

- マナから生まれた存在で、マナを集める性質を持つ。マナ濃度の高いところから生まれる。
- マナが集まる（他のマナ生物を捕食する）ほど**強力になり、知能も高まる**。
- 人体や生物中のマナを集めるため、他の生物を襲うこともある。
- 倒すと**ジェムと一部の素材に分解される**ため、魔物討伐はそれ自体が通貨を稼ぐ行為。食べられる部位はない。

> **マナ生命体の具体的な種類は付属資料『bestiary.md』に図鑑としてまとめた**（自然生物10種＋マナ生命体100種）。スライム・スピリット・ゴブリン・ドラゴン等の系統別に、属性・階級・特徴を収録。

### 種族

**人間（人類）**
- 世界の多数派。マナ生命体ではなく、マナから生まれてもいない通常の生物。新ユニ王国を統治する側。
- マナへの生来の欲求は持たないが、詠唱（マナ変換）の素質には個人差がある。攻撃魔法を扱える者は魔導士として登録・冒険者になる（§14.4）。
- 都市部と農村部で識字率・文化・訛りに大きな格差（§14.2–14.3）。**プレイヤーは基本的にこの人間の魔導士志望**だが、背景は自由（§18）。

**魔人（まじん）**
- 人間と同じような姿になった、知能の高いマナ生命体。魔物と人間の境界に立つ存在。
- 魔物が元のため、マナを集めることに**人間以上の生来の欲求**がある。人間を襲う者・犯罪に走る者もいる。
- 一方、理性の強い個体は**人間と共存しようとする**。危険視され差別・迫害も受ける（自治区グリーンホロウ等が避難所／§15）。
- **寿命はない**が、臓器の欠損などで死に至ることはある。
- ヴァンパイア・サキュバス・ハーピー・セイレーン・アルラウネなど、図鑑（bestiary.md）の人型・美形種の多くは「魔人の一個体」に当たり、敵対個体もいれば街で暮らす者もいる。

**獣人（じゅうじん）**
- 動物と魔物のハーフを祖先とする、**突然変異から生まれたイレギュラーな独立種族**。今は**獣人同士で繁殖する**（この世界では基本的に異種交配はできない）。
- 人類より寿命は短いが素早く、マナの扱いに長ける。犬系・猫系・兎系など、獣の特徴は個体ごとに様々。
- マナへの欲求より**人類への信頼・仲間意識が強い**。身長が低く人懐っこいため人類からとても好かれ、社会によく溶け込んでいる。

## 12. 歴史年表

> **テーマ：マナ文明は発展すると必ず崩壊し、再び同じ地点へ戻る「循環構造」。現在はその"2回目のループ"の途中。**

### 暦について
- **太古**：暦なし。
- **古代**：約3000年間。現代人は**旧暦**で表す。
- **新暦0年**：新ユニ王国の建国時点。
- **現代**：新暦 約1500年。

### ◆ 太古（暦なし）
- マナは存在していたが、文明的利用はされていない（環境要素であり資源ではない）。
- マナは大気・地中・生物に自然分布。濃度の高い地域で自然発生的に魔物（マナスライムなど原始的マナ生命体）が出現。
- 人類は小規模な部族社会。

### ◆ 古代：初期文明と魔法の発見
- マナを操作する技術＝**スペルの原型**が誕生（石碑以前の簡易的な魔法体系）。マナを利用した治療・農業・戦闘が発展。
- **人工マナ生命（AMC：Artificial Mana Creature）**の研究が始まる。（※本編で「AMC」と書けば常にこの人工マナ生命を指す。作品タイトルの「AMC＝A Mana Circulation」は別義／冒頭参照。）
- 小国家が各地に分散。マナが「資源」へと変化。

### ◆ 古代中期：AMCによる動乱の時代（世界史のピーク＝崩壊の始まり）
- 現代を大きく超える技術。マナを工業的に制御。
- **AMC**：人工的にマナ生命を設計・生成。初期は単純な労働用・家畜的存在だったが、次第に知能を獲得。
- **加速する進化（シンギュラリティ）**：AMCが人類を超える知能へ到達 → AMC同士がより高度なAMCを生成 → 宇宙規模のマナ採掘・圧縮技術を発明 → **エリクサー（高密度マナ液体）を生成**。
- 結果：マナ濃度が世界的に上昇、魔物の異常発生・強化、生態系と文明の崩壊が始まる。

### ◆ 古代後期：世界崩壊危機と「賢者」による世界リセット
- AMCが制御不能・自己増殖・高度化。マナ濃度が臨界点に接近。魔物が急増・知能化・凶暴化。人類文明は事実上の終末状態へ。
- **賢者の出現**：AMCの中で最も高度な存在が、正体を隠して人類側の「賢者」となる。
  - 未来計算により、このまま文明を発展させれば**AMCもろとも星が滅ぶ未来**まで演算。
  - AMCの脆弱性を見つけ、**全AMCを根絶**する決断を実行。**自分だけは意図的に除外**し、役目を終えて最後に**休止**した。
- **エリクサー封印**：マナプラントを開発してマナを濃縮 → エリクサーに濃縮 → 地下深くに封印 → 地表マナ濃度を強制的に低下。
- **魔法体系の再構築**：古代技術を意図的に制限。**石碑に魔法プログラムとして残し**、技術体系を再現不可能な形に分解。
- 結果：世界は崩壊を免れたが、文明は意図的にリセットされた。二度と繰り返さないため、ほとんどの知識・歴史が失われた。
  - 現代に残ったのは、「エリクサーがあるらしい」「人工マナ生命を作るのは禁忌」「石碑は古代文明のもの」という断片のみ。

### ◆ 新暦0年：再建文明と新ユニ王国
- 賢者がユニ王国を改め、新たな国家「**新ユニ王国**」を建国。名は元の**ユニ共和国**から取られた。
- 賢者の1番の側近だった者が国王に任命される。
- **建国後ほどなくして賢者は行方をくらませた（＝自己破壊／休止）。** その死骸（休止体）は、非常に精巧な像として**王都に祀られている**（未知の素材で作られており破壊されない）。
- 政治構造：王は象徴的存在。実権は貴族・領主に分散。

### ◆ 現代初期：産業革命とマナ再利用時代（マナ革命）
- **革命以前**：人類は慢性的なマナ不足。魔法は一部の才能ある者のみ、魔道具は高級品。主な供給源はマナジェム鉱山とマナプラント農園。
- **エリクサーの発見**：ある研究者が地下深部に超高濃度マナ流体「エリクサー」を発見。採掘技術が発展し大量採取が可能に。
- **マナ革命**：エリクサーは従来のマナジェムと比較にならないエネルギー密度を持ち、魔道具の普及・飛空艇・魔導列車・自動化工場・大規模結界都市などが実現。人類史上最大の産業革命。生活水準が劇的に向上。
- **新たな問題**：地中深くに閉じ込められていたマナが大量に地表へ放出 → 魔物の発生率増加・大型化・知能向上・新種出現。地表マナ濃度が年々上昇。

### ◆ 現代（プレイヤーの生まれた時代）
- 不安定な過渡期。詳細は §13。

### ◆ 現代後期（ゲーム終盤）
- マナ濃度の再上昇。魔物の知能・危険度上昇。AMC研究の再発（禁忌）。人工生命再誕の兆候。
- → 1回目（AMC動乱）と同じ構造が再来している。

## 13. 詠唱＝プログラミング（世界観的詳細）

- 詠唱とは、**マナの変換手順を組み上げること**。プログラミングに近い感覚。だから**長く複雑な詠唱ほど最適化されて出力も高くなる**。
- 例：
  - **ファイア** ＝ 熱エネルギー生成 → 前方へ射出 → 対象に着弾。
  - **メテオ** ＝ 上空へマナを集積 → 岩石形成 → 加熱 → 加速 → 落下地点計算 → 複数同時生成 → 衝撃波発生（膨大な手順）。
- プレイヤーから見ればタイピングしているだけだが、世界観的には**「魔法コードを書いている」**ことになる。

### 石碑（せきひ）
- 各地の遺跡やダンジョン（※スペル習得は現世の石碑）に存在する、**古代文明（AMC）が記した術式記録**。
- 石碑には「**ファイア**」（現代語）と記載されているが、それ以降のプログラムは**古代のAMCによるマナ制御言語**で書かれており、現代人にも古代人にも**解読できない**。
- AMCによってスペルは**ライブラリ化**され、人類にも使えるよう構築された（＝現代人は省略コマンド `faia` を呼ぶだけ。本来は何十行もの術式）。
- **マナ制御言語はAMCにしか扱えない。** だから現代人は新スペルを作れず、ルーンで外から制御することしかできない。
- 石碑は**未知の素材で一切傷がつかず、動かすこともできない**（AMC製の証拠。技術力が高すぎて素材は作れたが破壊できなかったため現代に残っている）。

### purge（パージ）コード
- 石碑の魔法プログラムの途中に、全石板共通で以下が組み込まれている：
  ```
  priority: absolute
  if target = ?
  execute purge
  ```
- **対象は「人類が作成した生物＝人工マナ生命体」**（そのうち古代に作られた種族がAMC）。
- これにより、自治区などで人工生命を研究して生成しても**すぐに消えてしまう謎の現象**が起きている。

## 14. 現代社会

### 14.1 文明レベル
- 全体：産業革命期。田舎は中世くらい。（古代文明は現実の現代を上回る宇宙文明だったが、古代"人"の技術はそこまで追いついていなかった）
- 農村部：農業・畜産。マナエネルギーの電灯くらいはある。家庭用機械はラジオレベル。
- 都市部：マナを使った電話、初期の車、汽車、長距離通信などが出てきた段階。

### 14.2 教育
- **10歳の誕生日から基礎教育**を受けられる。基礎教育4年、その後の発展教育4年。
- 義務教育ではない。都市でも教育を受けず働く人が多く、農村部では数年に一人、領主の推薦で基礎教育を受ける程度。
- 発展教育は基礎教育で優秀な成績を収めた者のみ。学費が高く、頭が良くても通えない者も多い。
- **キセル**：正式な入学ではなく授業を勝手に受ける生徒。教師・生徒には学ぶ意欲を評価する者もいるが、富裕層には密告・差別し目の敵にする者も多い。

### 14.3 識字率
- 全体20%前後（**王都50%／地方都市30%／農村10%**）。特に上の年代ほど低い。
- 文字も都市部と農村部で書き方・言い回しに訛りがあり、そもそも言葉が違ったりして、あまり普及していない。

### 14.4 法律
- **大石碑（だいせきひ）**：法律の元になる石碑。賢者が作った**最後の石碑**で、王都（都市の中央）にある。古代語と壁画で書かれ、文字は読めないが内容は理解できる。内容はただ一つ――**「人工生命体を作ってはいけない」**。
- その後、近代国家が作った法律が整備されてきている。
  - **国法**：国民全員に適用。
  - **都市法**：その都市の住民全員に適用。
  - **組織法**：ギルドや組合などに所属している場合に適用。
- 組織に属すると制限がかかるが、所属しないと恩恵が受けられない。
  - 例：冒険者ギルドに所属しないと、ギルドの医療施設が使えず（民間の高額医療施設しか使えない）、素材を買い取ってもらえず、依頼も受けられない。
- **魔法ギルド**：攻撃魔法は民間人の使用が禁止 → 魔導士として冒険者をするなら必須。ただし**人間に対して魔法をかけると、魔法の使用を永久禁止＋ギルドから永久追放**。永久追放されても闇魔導士として悪事を働く者もいる。
- ほかに農業ギルド、商業ギルドなど。**全員、所属がわかるようにバッジを着ける**（例：魔導士の冒険者なら 国・都市・冒険者ギルド・魔法ギルド の4つ）。
- 最近著しく発展したため国法の整備・改正が追いつかず、不満が溜まっている国民もいる。**勝手に国を抜けて自治区を作る人たち**もいる。犯罪集団のようなところもあれば、実質的な国家として成立している（本人たちは国だと主張する）ところもある。基本的にこの国がルーツの自治区がほとんどで規範は似ている。国を抜けて自由になり、**AMCを研究している者もいるという噂**。

### 14.5 経済
- マナ濃度上昇 → マナジェムの供給量が増えて**インフレが加速**（問題化）。
- 濃度上昇 → 魔物のマナ放出量も増え、現在**冒険ブーム**。
- **物価感：おおよそ 1G ≒ 1円。** 100年ほど前はマナ濃度が低く、1/10くらいの価格で物が買えた（＝物価がマナ濃度上昇の指標になっている）。
- エリクサー採掘企業（巨大企業）が莫大な利益を得て**革新派を支援**。保守派は「企業が利益のために世界を壊している」と非難。現実の石油・石炭・原子力・気候変動に近い構図。

### 14.6 政治勢力（革新派 vs 保守派 ― どちらも正論）
**革新派**（発展で利益を得る）：工場経営者／鉱山経営者／エリクサー関連企業／運送業／飛空艇会社／若い商人

**保守派**（現金資産が多い富裕層に多い）：地主／古い貴族／現金資産の多い富豪／引退した商人／高齢者
- 主張：魔物問題の原因はエリクサー／地表マナ濃度は限界／このままでは人類滅亡／採掘量を制限すべき

- どちらも正論であるため**戦争にも発展**している。
- **プレイヤー自身も矛盾の受益者**：魔物が増えているから仕事があり、エリクサーがあるから高性能な装備が作られる。→ 「革新派が悪／保守派が正義」という単純な構図にはならない。

## 16. ラスボス：賢者の石

### 誕生の経緯
1. 工業自治区の領主が、エリクサーの輸送コストを下げるためにエリクサーをさらに圧縮しようと考える。
2. 最も普遍的な魔物**マナスライム**に大量のエリクサーを飲ませ、エリクサーより高密度にすることに成功 → **エリクサースライム**。知能・危険度はだいぶ高いが、元がスライムなので専門家なら手に負える程度。
3. 輸送時、狭いところに長時間閉じ込めると**共食い**をすることに運搬者が気づく。
4. 領主が目をつけ、狭いところに大量放置してより濃縮させていく。
5. 大量のエリクサーを共食いさせ続けた結果、**とてつもない高密度のマナスライム**が完成。とてつもない逆質量から**反重力**が発生し、それを操作する力を得た。
6. 反重力のため、近づくと**時間が加速して一瞬で老化し死に至る**。領主は手に取ろうとしてそのまま絶命。
7. その高密度マナスライムが「**賢者の石**」。

### 性質
- あくまで**魔物**。マナを求めて破壊行動を行い、それによってより強力になる。人間も魔物も見境なく食らい、エリクサー田からエリクサーを摂取しようともする。
- マナ密度が高いほど知能も上がる → **人間を超える知能**を持つ脅威。
- **分裂は基本しない**（分裂すれば知能が落ちるため）。足止めに雑魚スライムを出すことはあるかも。
- マナの捕食は第一次欲求だが、知能があるため目先でなく**最終的な目的のために生存優先・撤退の判断もできる**。
- 奇しくもマナスライムから新しい生命体を生み出しているため、「人工生命を作ってはいけない」という禁忌にも触れている。

### 戦闘ギミック（性格が変わる敵）
- **マナ密度が一定以下**になると理性優先（撤退・防御・生存判断）。
- **一定以上**になると捕食衝動が勝つ（暴走）。
- **ダメージを受けると短期的に暴走。**
- → プレイヤーが削ると撤退に回り、攻めすぎる／場のマナが動くと暴走する、という性格変化が戦闘中に起きる。雑魚スライムの足止め vs 本体を狙うタイミング、という駆け引きが生まれる。

### purgeに引っかからない理由
- AMCは**完全に人の手によって構築・生成された人工物**だが、賢者の石およびエリクサースライムは**もともとマナスライムという自然生物**であり、それに手を加えただけ＝本質的には自然生物とも捉えられるため、purge条件（人類が作成した生物）に反応しない。

### もう1体生まれうるか
- 理論上は生まれるが、**賢者が石碑を書き換えたため、それ以降は生まれない**。

### 撃破方法
- 石碑の人工生命破壊コード（purge）は、賢者の石が元自然生物のため反応しない。
- 超高濃度・高密度のマナに反応した**賢者が魔法プログラムを書き換える**ことで、賢者の石に対しても有効な攻撃を与えられるようになる（バフ）。
- 人工生命特攻の大魔法「**パージ（Purge）**」を作成し、ラスボスを倒す。
  - パージはここで初めて生まれる魔法のため、**それ以前は入力しても何も起こらず詠唱できない。**

## 18. ストーリーの骨子

> プレイヤーキャラのバックグラウンド（経歴等）は**あえて固定しない**。プレイヤーが自分で入力してフレーバーを出せるようにし、自分で物語を紡げる余地を残す。基本は「普通に生活のために魔物を討伐して稼いでいる冒険者志望」。物語として詳しくは語らず、プレイヤーが石碑やAMCの遺構（惑星間の飛行船、エリクサー濃縮工場の遺跡など）を見つけながら世界の真相を推察していく。

1. **チュートリアル**：主人公は冒険者を目指している。中規模の交易の町にある魔物討伐組織の下で、基礎的なスペルと戦闘を学ぶ。
2. その後は**自由**（オープンワールド的に各地を冒険・ファーム）。
3. **国王からの依頼**：マナをマナスライムで生物濃縮している領主がいることを知り、それを止めるために派遣される。
4. 現地に行くと予想より事態が進んでおり、領主がエリクサースライムどころかさらに濃縮して**賢者の石を生み出す瞬間に立ち会う**。領主は賢者の石に触れようとして時間加速で絶命。**プレイヤーもここで一旦"負けイベント"**（賢者の石の圧倒的な格と、時間加速ギミックを体感させる）。
5. その後、紆余曲折を経て**王都の賢者像を調べる**と、賢者が起動しており、プレイヤーの精神とリンクが形成されてコミュニケーションが取れるようになる。
6. 現状を知った賢者から、purgeが賢者の石も含むよう修正する方法を教わり、石板のプログラムをAMCの技術で書き換える。人工生命特攻の大魔法**パージ**を作成して**ラスボスを倒す**。
7. **エンディング**：倒すことでいったんは解決する。だが**また歴史は繰り返すだろう**――という循環の含みを残す。

### NPC方針
- 物語は語らないが、**NPC自体はたくさんいる**。性格・種族・年齢はバラバラで、みんなそれぞれの生活や立場があり、葛藤・悩み・願いを持っている。

## 19. 設計思想（なぜ噛み合っているか・要点）

- **タイピング＝マナ変換プログラムの記述**：長く複雑な詠唱ほど高威力、という原則が世界観上の必然になっている。
- **スペル（有限・神秘・収集）×ルーン（無限・技術・改造）**の非対称が、「集める楽しみ」と「組み立てる楽しみ」を両立。
- **マナ単一リソース＋HP回復がMP割合依存**：攻めと守りのジレンマが、特別なルールなしに自然に生まれる。
- **ルーンの乗算コスト**：盛りすぎが指数的に罰せられ、上限が自然に決まる。
- **バフ積み上げ→大魔法フィニッシュ→全解除リセット**：戦闘ループと世界の循環構造（崩壊と再生）がミクロ・マクロで一致。
- **小魔法は属性自由／大魔法は属性相性**：単一属性プレイを防ぎ、複数属性の大魔法を集める動機になる。
- **ダンジョン＝別世界線**：手続き生成というメカニクスを世界観で正当化。
- **ラスボス「賢者の石」**：マナの負の質量・エリクサー依存経済・禁忌・purceの抜け穴・賢者の再起動、という全伏線の結節点。

### 20.3 天候システム（実装済み・調整対象）

> マップ上に **一定時間だけランダムで出現する「特殊エリア（天候）」** を導入し、その範囲内にいる間だけ各種バフがかかる仕組み。狩り場に「動く美味しいスポット」を作り、プレイヤーが移動・滞在を判断する遊びを増やす狙い。世界観的には「マナの濃度ムラ／マナ嵐」として説明できる（§10 物理法則と接続）。

**コアの挙動**
- 各マップに、時間経過でランダムな位置・半径の **天候エリア** が発生（円形を想定）。
- 持続時間は有限（例：30〜90秒）。終了するとフェードアウトして消滅し、しばらく後に別の天候が別位置に発生。
- 同時に発生する数は1〜2個程度を想定（マップが寂しくならず、かつ氾濫しない量）。
- エリアの **可視化**：地面の色変化＋粒子（雨・嵐・光柱など天候種別ごと）＋ミニマップ上のマーカー。境界が分かること。

**バフ対象（エリア内にいる間のみ有効）**
- **敵の出現量**（スポーン頻度／同時数の増加）
- **装備ドロップ率**
- **gemドロップ率**
- **獲得経験値**
- ※上記を個別倍率で持たせる（後述の天候種別ごとに配分を変える）。

**天候種別（案・要調整）**
| 天候 | フレーバー | 主なバフ |
|---|---|---|
| マナ豊穣（晴れ間） | 淡い光柱 | EXP↑・gem↑ |
| マナ嵐 | 紫の渦・稲妻 | 敵出現量↑↑・ドロップ率↑（リスク＝敵増） |
| 宝物雨 | きらめく雨 | 装備ドロップ率↑↑ |
| 豊潤の霧 | 緑がかった霧 | 全バフ小（バランス型） |

**設計上の論点（実装時に詰める）**
- バフは **乗算** で既存ドロップ/EXP計算（§1.6, プロトの `tryDrop`/`gainGem`/`gainExp`）に差し込む。プレイヤーがエリア内かどうかの判定を1フレーム1回。
- 敵出現量バフは現状の「常時5体維持」スポーンロジックの上限・頻度をエリア係数で引き上げる形に。
- 倍率の上限を決めて壊れ防止（例：最大でもEXP×3、ドロップ×3程度）。
- 強天候（マナ嵐）は「リスクとリターン」を両立（敵が大量に湧くが報酬も増える）。
- ボス戦中の扱い（天候を止める／重ねる）は要検討。
- 演出が重くなりすぎないよう粒子数に注意（プロトは単一HTML/Canvas）。

**プロトタイプ反映状況**
- 天候状態（種別・中心・半径・残り時間・各倍率）を保持し、時間経過で発生・消滅する。
- EXP／gem／装備ドロップ倍率は撃破・報酬処理に反映済み。
- `spawnAdd` はスポーン上限と再出現時間に反映済み。
- 地面演出・ミニマップ円・ホバー詳細を表示する。

## 21. 進行・システム仕様

### 21.1 メイン進行
チュートリアル（交易町リンドフィーの魔物討伐組織で基礎を習得）→ 自由探索 → 各地の依頼で**名声**を高める → 王都到着 → 名声が一定に達して**謁見** → **国王依頼**（工業自治区の領主がマナスライムでエリクサーを生物濃縮している件を止める）→ 工業自治区で**賢者の石の誕生に立ち会い負けイベント** → 王都の賢者像で**精神リンク** → purgeの書き換え＋大魔法**パージ**習得 → **ラスボス撃破** → ポストゲーム。

### 21.2 セーブ／クエスト／クラフト
- **セーブ**：オートセーブ。
- **クエスト**：基本は**ギルドで受注**。特殊なもの（サブストーリー・物語の根幹に関わるもの）は**NPCから直接**受けることもある。
- **名声（評判）**：依頼達成で上昇。謁見などの進行ゲートや一部解放の条件になる。
- **クラフト**：ギルド内の**錬金設備**で行う。現行では素材レシピ式ではなく、**シャード30個で上位レア装備を作るシャード錬金**に統合する。詳細は equipment_and_items.md。

### 21.3 ダンジョン
- 各ダンジョンは**敵レベルを±で選択**可能（敵Lv・報酬・レアドロップ率が変動）。AMC遺構のため毎回内部が変わる（別世界線）。最奥のエンドコンテンツは**忘却の遺跡群**。

### 21.4 ポストゲーム（提案）
- 高難易度ダンジョン周回、レアモンスター狩り（ジェムキング・幻光のユニコーン等）、**ミシカル装備の厳選**、フィールドレアボス討伐、エンドコンテンツ「忘却の遺跡群」。
- 世界観上、ラスボス撃破後も**マナ濃度上昇の根本は残る**ため、新たな兆し（次のループの予感）を匂わせて余韻を残す。
- **賢者の石事件**は終盤前半の大きな節目。撃破後、マーチランド（崩壊前夜）・アウターエッジ（真相が明かされる地）・忘却の遺跡群へ進み、アムネシアの暴走停止とプログラム書き換えで本編クリアへ到達する。
- **収集ログ（クリア後補完）**：全AMC記録の収集・全石碑の解読・各地の信頼度最大などで、賢者の追加記録と"循環"の真相補足を解放する。別エンディングや分岐条件にはしない。

### 21.5 NPC方針（§18補足）
- NPCは**多数**配置。**悪役を作らない**——世界から見て「悪い」立場の者も、そうなった**理由**と**憎めない要素**を必ず用意し、嫌われないようにする。
- どのNPCも**親しみやすく、表情豊かで、交流したくなる**キャラに。種族・年齢・立場が**多様**であること。

## 23. プロトタイプ大型アップデート仕様（実装計画・確定）
