# スペル & ルーン 資料（Spells & Runes）

> 本編設定資料（game_design_bible.md）の付属資料。魔法システムの**詳細リファレンス**。
> 概念・構文・乗算コスト・バフの積み上げ・属性と相性などの**ルール本文は本編 §2** にある。本ファイルは「共通パラメータ」「各スペルの数値＋バフ」「ルーン全一覧」を担当する。

---

## 1. 前提ルール（要約）

- **構文**：`[ルーン…複数・順不同] + [スペル…最後に1個]`（例 `double ranged faia`）。
- **ルーンの乗算コスト**：消費マナはルーンで乗算（例 ×2×1.5×2＝6倍）。盛るほど指数的に重く、詠唱文字列も伸びる。**同種ルーンの重複は不可**。
- **バフの積み上げ**：スペルは「メイン効果＋バフ効果」を持つ。
  - **同じスペルのバフは重ねがけ不可**＝効果時間のリフレッシュのみ。
  - **異なるスペルのバフは同時維持できる**（数の上限なし。ただし継続時間＋詠唱時間で自然に頭打ち）。
  - **大魔法はバフを持たず、撃つと全バフを解除**する（フィニッシャー専用）。
  - **バフはMP消費を増やさない**（増やすのはルーンのみ）。
- 数値はすべて**暫定（バランス調整フェーズで確定）**。

## 2. スペルの共通パラメータ

すべてのスペルは以下を共通で持つ。これにより**どのルーンもどのスペルに必ず効く**（n×mの組み合わせを個別実装なしで成立）。

| パラメータ | 意味 |
| --- | --- |
| **威力** | 命中時／範囲内の敵に与えるダメージ |
| **範囲** | 効果範囲（半径）。単発スペルでも、命中時の小さな着弾・爆発範囲として必ず持つ |
| **距離** | 射出後、敵に当たらなかった場合に消えるまでの飛翔距離 |
| **継続時間** | 効果が持続する時間（弾の飛翔時間、設置物・継続エリアの存続時間など） |
| **MP消費** | 基礎マナ消費（ルーンで乗算される前の値） |
| **詠唱時間** | 確定から発動までの時間 |

※拡張パラメータ（弾数、状態異常の蓄積量、ノックバック、弾速など）も必要に応じて持つ。

**例：ファイア**（単発でも全パラメータを持つ）：着弾で**小爆発**。範囲＝爆発半径／威力＝爆発ダメージ／距離＝当たらなければ消えるまでの飛距離／継続時間＝飛翔時間。
→ `ワイド`で爆発拡大、`レンジド`で遠くへ、`ロング`で長く飛び、`パワー`で爆発威力増。

## 3. バフ効果の設計

各属性の方向性（§2.7）に沿ってバフを割り当てる。

| 属性 | 方向性 | 主なバフ |
| --- | --- | --- |
| 火 | 火力 | 威力+／クリティカル率+／クリティカルダメージ+／火傷蓄積+ |
| 風 | 詠唱速度 | 詠唱時間-／移動速度+／回避率+ |
| 水 | 再生・支援 | HP自動回復+／MP自動回復+／MP消費-／被ダメージ-／鈍化蓄積+ |
| 土 | 防御 | 防御力+／被ダメージ-／気絶蓄積+／ダメージ吸収バリア／気絶耐性 |

> **【重要】効果量と継続時間は反比例**：弱いスペルほどバフの効果量は小さいが**継続時間が長い**（土台）。強いスペルほど効果量は大きいが**継続時間が短い**。これにより弱い魔法→強い魔法へ自然にエスカレートし、短命な強バフが切れる前に**大魔法でフィニッシュ**する流れが生まれる。
> 目安：基礎スペル ≈ 38–40s／初級 ≈ 30–32s／中級 ≈ 22–28s／上級 ≈ 14–18s。バフ専用・回復スペルは土台として長め（28–35s）。
>
> **状態異常蓄積+系（火傷/鈍化/気絶など）**は敵に対する蓄積を早めるバフ。攻撃の手数で積み、大魔法のフィニッシュ前に弱点・状態異常を整える流れになる。
> **バランスTODO**：異なるスペルでも同カテゴリのバフ（例：威力+を複数）が無制限に乗算すると過剰になりうる。同カテゴリは上限／逓減を設けるかは調整フェーズで検討。

## 5. ルーン一覧

> 効果・消費マナ倍率は暫定。乗算で累積（§2.4）。同種ルーンの重複は不可。すべて§2の共通パラメータ上で動くため、原則どのスペルにも適用できる。

### パラメータ調整系
| ルーン | 効果 | MP消費 |
| --- | --- | --- |
| パワー | 威力 ×2 | ×2 |
| ワイド | 範囲 ×1.5 | ×1.5 |
| レンジド | 距離 ×2 | ×1.3 |
| ロング | 継続時間 ×2 | ×2 |
| クイック | 詠唱時間 ×0.5 | ×1.5 |
| エコ | MP消費 ×0.7（威力 ×0.7） | —（威力を犠牲に効率化） |

### 挙動変化系
| ルーン | 効果 | MP消費 |
| --- | --- | --- |
| ダブル | 一定時間後、修飾したスペルをもう一度発動 | ×2 |
| トリプル | 同上を計3回 | ×3 |
| ホーミング | 弾が敵を追尾する | ×1.5 |
| ピアース | 弾が敵を貫通（最初の命中で消えない） | ×1.5 |

例：`ホーミング アイシクル`＝追尾する氷柱／`ピアース ロック`＝敵を貫く岩／`チェイン サンダー`／`ダブル ワイド メテオ`（高コストの極大連発）。

## 6. 注記
- **大魔法はバフ非所持＋全解除**のため、上の一覧でバフ欄が「—」。フィニッシャーとして使う。
- バフ専用スペル（ヘイスト・フライング・各種シールド系等）は威力が「—」で、積み上げ・支援に特化。
- 各スペルの正確な数値、状態異常蓄積の具体量、ルーン適用時の詳細挙動は実装フェーズで確定。

## 大魔法の再設計（確定）
- **基礎ダメージを引き下げ**、小魔法でバフを積んでから撃つフィニッシャー設計。
- 大魔法は **小魔法バフを×3**、**ルーン効果を×2**（powMul等の1からの偏差を2倍）で受ける。
- **消費MPに応じた追加補正**（+最大60%）。
- 当てやすさでダメージ調整：自動着弾AoE（メテオ系=当てやすい→低め）＜自分周囲AoE（中）＜投射/着弾爆発（要狙い→高め）。
- 発動後は **全バフ解除**（既存仕様）。
- バリエーション：メテオ/タイダル/ストーム/ライトニング＝対象着弾AoE、ダイアモンドダスト/ガイア＝自己周囲AoE、ジャベリン＝貫通投射、**アイシクル＝着弾爆発投射**。

### 新規・水大魔法「アイシクル」
氷球を前方へ発射し、着弾地点で大爆発（splash170）。水属性ダメージ＋鈍化（slow4）蓄積。大魔法のため発動後に全バフ解除。読み：あいしくる。

## 今後追加予定の魔法カテゴリ（次フェーズ）
魔法種を約2倍に拡張。多段ヒット、扇形範囲、設置型ダメージエリア、自己周囲ダメージエリア、ファンネル系（追従設置砲台）、MP自動回復強化、防御・補助系を追加。小魔法＝バフ/補助/削り、大魔法＝フィニッシュで役割分担。

## 入力・演出仕様（次フェーズUI）
- 自キャラの攻撃も **入力完了時点で方向・地点を固定**（敵と同様）。攻撃予測を **青色** で表示。
- 投射の **飛翔速度に変化** を付ける（呪文ごとに speed 差／ルーンで飛距離可変）。
- **Mキーで魔法・ルーン一覧**：魔法は 名前/属性/小・大/消費MP/威力/詠唱時間/バフ効果、ルーンは 効果/ダメージ倍率/MP倍率/範囲/飛距離/発動回数 を表示。
